宣告 tiles[100][100] 陣列 用來存已經放置的磁磚位置
m[i] 代表 第 i 組磁磚有幾塊
w[i] 代表 第 i 組磁磚 每一塊的寬
h[i] 代表 第 i 組磁磚 每一塊的高

可利用 DFS 將 tiles 的每一格遍歷，並一一決定
該位置應該放置 什麼樣的磁磚

DFS 流程:
	param: x 代表要填入的格子該格的高度
	param: y 代表要填入的格子該格的高度

	if (y >= width) x+=1 y=0           // 該 x 列的所有可放的格子已填滿，換到下一列
	if (x >= height) ways+=1 return    // tiles 所有可放的格子已填滿 ways += 1，並結束該層遞迴
	
	if (!tiles[x][y]) // tiles[x][y] 為空
		for 跑 所有的 i
		並同時檢查 (h[i], w[i]), (w[i], h[i]) 能否放入空格
		可以的話進入下一層遞迴
		不行的話則 continue
	else              // tiles[x][y] 不為空
		直接進入下一層遞迴

由於題目有規定，所有的磁磚都會用到，因此不用額外做判斷
註: 磚塊可旋轉，因此 DFS 時，每一塊可能需要判斷兩次
